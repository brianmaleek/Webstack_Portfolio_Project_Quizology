About Quizology

Welcome to Quizology

Quizology is a web application that allows users to take multiple-choice quizzes on various topics.

This application is built using Django, a high-level Python web framework, and incorporates features such as user authentication, session management, and database storage of quiz results.

We hope you enjoy using Quizology to test your knowledge and learn new things!
